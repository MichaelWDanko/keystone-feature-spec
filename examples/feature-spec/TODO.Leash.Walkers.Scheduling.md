# Leash.Walkers.Scheduling

Defines intended scheduling behavior that has not yet been implemented in the
fictional Leash dog-walking app.

## Requirements

- A walker MUST be able to define recurring available hours.
- An owner MUST NOT be offered a walk outside a walker's available hours.
- Changing future availability MUST NOT alter a confirmed walk.
